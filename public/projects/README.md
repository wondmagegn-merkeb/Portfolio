## Image Requirements

- Format: JPG, PNG, or WebP
- Main image: 800x600px or similar aspect ratio
- Gallery images: 1200x800px or similar aspect ratio
- Optimize images for web (compress to reduce file size)

## Adding More Projects

For additional projects, add images following the same naming pattern:
- `project-name-main.jpg`
- `project-name-1.jpg`
- `project-name-2.jpg`
- etc.
